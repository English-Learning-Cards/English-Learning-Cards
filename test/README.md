## 🎯 テスト協力のお願い

### 特にテストしてほしい環境
- iPhone (Safari / Chrome)
- iPad (Safari / Chrome)
- Android (Chrome)

### テスト手順
1. `voice-debug-v1.html` を開く
2. 🍎 apple テストボタンを押す
3. 英語発音かカタカナ読みかを確認
4. 結果をIssueで報告

### 報告項目
- **端末**: iPhone 14 Pro
- **ブラウザ**: Safari 17.1
- **音声結果**: 英語発音 / カタカナ読み
- **音声状態**: ✅準備完了 / ⚠️エラー
